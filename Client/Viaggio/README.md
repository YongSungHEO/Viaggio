## Viaggio - 여행 일기

## 주요 기술 및 설명

##### android
+ 차세대 안드로이드 언어 Kotlin으로 모든 코드 작성
+ Retrofit2, OKHTTP3을 사용하여 웹서버와 통신
+ MVVM Pattern을 사용

##### web
+ 파이썬의 웹 프레임워크인 Flask 사용
+ MVC Pattern을 사용
